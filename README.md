# GFN Fantasy League 🏈

Historical stats dashboard for the GFN Fantasy Football League (2011–2025).

**Live site:** https://yourusername.github.io/gfn-fantasy

---

## Project Structure

```
gfn-fantasy/
├── index.html                  # Frontend dashboard (GitHub Pages serves this)
├── fetch_gfn_data.py           # Data pipeline — run locally only
├── requirements.txt            # Python dependencies
├── .env                        # Your secrets — NEVER committed
├── .env.template               # Credentials template (safe to commit)
├── .gitignore
└── data/
    └── dashboard_data.json     # Generated by pipeline, committed to GitHub
```

---

## Local Setup

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.template .env           # Fill in your Yahoo credentials
```

## Running the Pipeline

```bash
python fetch_gfn_data.py --dry-run   # Test auth
python fetch_gfn_data.py             # Full fetch (first run ~5-10 min)
python fetch_gfn_data.py --year 2025 # Update current season only
```

## Weekly Update Workflow

1. `python fetch_gfn_data.py --year 2025`
2. In VS Code: stage `data/dashboard_data.json` → commit → push
3. GitHub Pages updates automatically within ~60 seconds
